During this semester I have been taking an information technology class which is currently requiring me to create this blog.
I decided to create the blog using a simple HTML page, wrapped in bootstrap with a modal, and am currently editing this on a manual index.html created on GitHub text editor.
This could easily be made very functional, if blog post information was stored in an SQL server, and fetched and formatted dynamically.
Neither the less, I am indeed enjoying this class, as well as the online class formatting. I do wish for more complexity in lessons, but I do enjoy attempting to do my best with the projects.
